POST /api/tron/watch-addresses

新增 Tron 监控地址

支持单个地址或批量地址写入。地址必须是合法的 TRON Base58 地址，即 T 开头。 已存在的地址不会重复插入，只会在服务日志中记录为重复地址。
请求体
{
  "address": "TXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX"
}
批量请求体
{
  "addresses": [
    "TXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX",
    "TYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYY"
  ]
}

请求说明

  • 只支持 POST。
  • 请求头必须带 Content-Type: application/json。
  • 如果配置了 web.api_key，则必须带 API Key。
  • 同一请求中的重复地址会自动去重。
  • 写入成功后会立即刷新内存地址缓存。
成功响应
{
  "success": true,
  "message": "ok",
  "count": 2,
  "addresses": [
    "TXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX",
    "TYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYY"
  ],
  "invalid_addresses": []
}
错误响应
{
  "success": false,
  "message": "no valid addresses",
  "invalid_addresses": [
    "bad-address"
  ]
}

curl 示例

curl -X POST '{{ .BaseURL }}/api/tron/watch-addresses' \
  -H 'Content-Type: application/json' \
  -H 'X-API-Key: your_api_key' \
  -d '{
    "addresses": [
      "TXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX",
      "TYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYY"
    ]
  }'

POST /api/refresh-addresses

通用刷新地址余额

支持 tron 和 bsc 两条链，支持单地址和批量地址刷新。请求体中的 chain 必填，批量最多 100 个地址。
请求体（单个地址）
{
  "chain": "tron",
  "address": "TXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX"
}
请求体（批量地址）
{
  "chain": "bsc",
  "addresses": [
    "0x1111111111111111111111111111111111111111",
    "0x2222222222222222222222222222222222222222"
  ]
}
成功响应
{
  "success": true,
  "message": "BSC 地址余额批量更新成功 2 / 2",
  "chain": "bsc",
  "address": "0x1111111111111111111111111111111111111111",
  "addresses": [
    "0x1111111111111111111111111111111111111111",
    "0x2222222222222222222222222222222222222222"
  ],
  "total_count": 2,
  "success_count": 2
}
错误响应
{
  "success": false,
  "message": "addresses count cannot exceed 100",
  "chain": "bsc"
}

curl 示例

curl -X POST '{{ .BaseURL }}/api/refresh-addresses' \
  -H 'Content-Type: application/json' \
  -H 'X-API-Key: your_api_key' \
  -d '{
    "chain": "tron",
    "addresses": [
      "TXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX",
      "TYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYY"
    ]
  }'

GET /api/tron/balances

Tron 余额列表

按分页列出 Tron 监控地址余额（TRX/USDT）。支持排序。
查询参数
  • address：指定地址（可选，传入时返回单条记录）
  • limit：每页条数（默认 20，最大 200）
  • offset：偏移（默认 0）
  • sort：usdt_desc/usdt_asc/trx_desc/trx_asc（可选）
curl 示例
curl '{{ .BaseURL }}/api/tron/balances?limit=20&offset=0&sort=usdt_desc' \
  -H 'X-API-Key: your_api_key'

GET /api/tron/transfers/in
/api/tron/transfers/out

Tron 转入 / 转出流水查询

按监控地址查询转入或转出流水，支持分页与时间范围过滤。返回字段包含 watch_address（命中监控的地址）。
查询参数
  • watch_address：监控地址（必填，Tron Base58）
  • asset_code：TRX / USDT（可选）
  • start_time：起始时间（毫秒时间戳或 RFC3339，可选）
  • end_time：结束时间（毫秒时间戳或 RFC3339，可选）
  • limit：每页条数（默认 20，最大 200）
  • offset：偏移（默认 0）
curl 示例
curl '{{ .BaseURL }}/api/tron/transfers/in?watch_address=TXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX&asset_code=USDT&start_time=2026-07-06T00:00:00%2B08:00&end_time=2026-07-07T00:00:00%2B08:00&limit=20&offset=0' \
  -H 'X-API-Key: your_api_key'
GET /api/bsc/transfers/in
/api/bsc/transfers/out
